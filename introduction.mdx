The USM Fitness API powers the gym management system for Universidad Santa María. It provides endpoints for student registration and login, time block scheduling, gym reservations, training plan management, and physical assessments — all secured with JWT authentication and a three-tier role system.

## How it works

<Steps>
  <Step title="Register and log in">
    Students register with their RUT and university credentials. All users authenticate with JWT tokens issued at `/api/login/`.
  </Step>
  <Step title="Browse available blocks">
    The API auto-generates the weekly schedule every Monday. Students see upcoming blocks with real-time available capacity.
  </Step>
  <Step title="Make a reservation">
    Students book one block per day. The system enforces capacity limits, with overflow allowed for sports-selected athletes.
  </Step>
  <Step title="Attend and track progress">
    Trainers mark attendance after each session. Students can view their training plans and physical assessment card at any time.
  </Step>
</Steps>

<Note>
  The API also exposes interactive Swagger documentation at `/swagger/` on any running instance.
</Note>
